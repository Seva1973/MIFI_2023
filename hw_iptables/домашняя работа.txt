Задание

Напишите правило, которое будет запрещать входящие ICMP echo запросы к серверу. После чего заскриньте вывод цепочки так, чтобы новое правило попало в скриншот, а также в скриншотах должна быть команда, которую использовали.

команды с учебных материалов
iptables -A INPUT -p tcp --dport 22 -j DROP

iptables -A INPUT -p tcp --dport 22 -j REJECT

Хост - машина с Win 11 c Ubuntu (WSL) "На борту" . ssh-Сервер - Linux Mint на виртуальной машине


команда для блокирования запросов на сервере, вводимая через ssh соединение с хоста на сервере

sudo iptables -A INPUT -p icmp -j DROP

команда снятия блокировке на сервере (подключились повторно через ssh)

sudo iptables -D INPUT -p icmp -j DROP
